Project Title Containerization of Apache Hadoop cluster and container monitoring using web portal.
Project description Apache hadoop has so many components but you have to containerize only HDFS with its operations and monitoring. All containers must be easily manageable from the Web portal so that all users can handle containers without knowing about them.
Technology used
  • Ubuntu 18.04 or later || centos 8.3 or later
  • Python 3.8 or later
  • Web port - HTML5 and css 3 for front end design
  • Back end will be python django or python flask or python CGI
  • Apache hadoop 3.x / 2.x
  • Open JDK 8 or later
  • AWS cloud Ec2 and EBS
  • Mysql or Mariadb database
  • Docker Engine 19 or later
